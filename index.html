Create a file setup.json in the same directory.
Put an array of the currencies you own, and configure id fields according to CoinMarketCap API.
Set your local currency by picking the language tag of your country. 

{
  "localCurrency": {
    "id": "USD",
    "languageTag": "en-US"
  },
  "currenciesOwned": [
    {
      "id": "bitcoin",
      "amount": 0.04151
    },
    {
      "id": "ethereum",
      "amount": 1.194119638
    },
    {
      "id": "ripple",
      "amount": 1060.17
    },
    {
      "id": "stellar",
      "stellarAccountId": "GCVQMOB6ASUZUJQ3EGQI7WOOIQ6HI6MVPMQCUO5NJYMXIFW3UUF4LM3C"
    }
  ]
}

Press F11 to enter presentation mode.
