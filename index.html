Create a file setup.json in the same directory.
Put an array of the currencies you own, and configure id fields according to ticker symbols on CryptoCompare.
Set your local currency by picking the language tag of your country. 

{
  "localCurrency": {
    "id": "USD",
    "languageTag": "en-US"
  },
  "currenciesOwned": [
    {
      "id": "BTC",
      "amount": 0.04151
    },
    {
      "id": "ETH",
      "amount": 1.194119638
    },
    {
      "id": "XRP",
      "amount": 1060.17
    },
    {
      "id": "XLM",
      "stellarAccountId": "GCVQMOB6ASUZUJQ3EGQI7WOOIQ6HI6MVPMQCUO5NJYMXIFW3UUF4LM3C"
    }
  ]
}
